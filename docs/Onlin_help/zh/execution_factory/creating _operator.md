# 如何创建算子
## 概述
算子是在数据流中对数据进行转换、计算或处理的独立功能单元。这些算子通过组合连接，形成了从原始数据到最终结果的完整、自动化的处理流水线。

## 创建算子
支持通过**函数计算**（自定义代码编写）和**算子编排**（可视化编排已有算子）两种方式创建自定义算子，创建后可被数据流模块引用。

### 关键术语
| 术语 | 含义 |
| ---- | ---- |
| 函数计算 | 通过编写 Python 代码实现自定义逻辑，支持代码编辑与调试 |
| 算子编排 | 通过拖拽可视化界面编排已有算子，实现组合式业务处理 |
| 调试 | 输入测试数据运行算子，验证代码 / 逻辑正确性的功能 |
| 超时时间 | 避免单个任务耗时过长导致系统阻塞的时间限制，可自定义配置 |

## 前提条件
1. 已登录 ADP 系统，且具备**算子管理**的新建权限
2. 若选择**函数计算**方式，需掌握基础 Python 代码编写能力

## 场景1：新建函数计算类型算子
### 操作步骤
1. 进入算子列表页：进入「算子工厂」>「算子管理」>「算子」页面
2. 发起新建操作：点击页面右上角【新建算子】，在弹窗中选择【函数计算】
3. 编写代码逻辑：
    - 进入「代码编辑」标签页，在编辑区按示例格式编写 `handler` 函数（需包含 `event` 参数，返回处理结果）
    - 参考页面注释补充逻辑（例：通过 `event.get` 获取输入参数，定义返回格式）
4. 配置元数据：
    - 切换至「元数据」标签页，在“基础配置”中填写：算子名称（必填）、算子描述（必填）、超时时间（默认 3000ms，可修改）
    - 在“输入参数”/“输出参数”区域点击【添加参数】，填写：参数名称（必填）、参数说明（必填）、类型（下拉选择）、必填状态（必填 / 非必填）
5. 调试验证：
    - 在右侧“调试”区域输入 JSON 格式测试数据
    - 点击【运行】，查看“输出”区域结果，验证逻辑正确性
6. 保存算子：确认配置无误后，点击页面右上角【保存】

### 注意事项
- 算子创建方式（函数计算 / 算子编排）选择后不可修改，需更换方式需删除后重新创建
- 元数据的入参 / 出参需与代码逻辑严格对应，否则会导致智能体调用时参数不匹配
- 调试数据需为合法 JSON 格式，超时时间需合理（避免过短导致执行中断）

## 场景2：新建算子编排类型算子
1. 打开新建页面：进入「算子工厂」>「算子管理」>「算子」页面
2. 发起新建操作：点击页面右上角【新建算子】，在弹窗中选择【算子编排】
3. 配置流程节点：
    - 设置“开始算子”，配置触发参数
    - 选择“执行操作”，选择各种执行操作并配置相关参数
    - 设置“结束算子”，定义输出结果格式
4. 验证运行：点击【运行】测试组合算子，通过“详情”查看运行状态；若失败，可查看错误日志定位问题
5. 调用使用：在数据流创建中，将该算子作为执行节点被使用

## 常见问题
### Q1：调试函数计算算子时，点击【运行】无输出结果怎么办？
A1：按以下步骤排查：
1. 检查代码中是否正确编写 `handler` 函数的返回逻辑（需确保有 `return` 语句）
2. 验证测试数据是否为合法 JSON 格式（可使用在线 JSON 校验工具检查）
3. 确认超时时间是否过短（可适当延长后重试）

### Q2：保存算子后，能否修改创建方式（函数计算 / 算子编排）？
A2：不能。创建方式在新建算子时已确定，保存后无法修改，若需更换方式，需删除后重新创建。
