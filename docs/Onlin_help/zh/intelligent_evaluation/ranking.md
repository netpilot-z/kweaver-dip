# 榜单

## 概述
榜单是基于统一测试指标与评测规则，对参测对象的测试结果进行排序与展示的结果视图。

**适用用户**：ADP 管理员、数据分析师、测评任务执行者


## 使用说明
- 榜单仅展示**已发布**的效果测评配置所对应的任务结果；
- 未发布配置下的测评任务结果暂不纳入榜单展示范围。


## 前提条件
在使用**榜单**功能前，请确认已完成以下准备：
1. 已登录 ADP，且具备**智能测评**模块的访问权限（如无权限，请联系平台管理员申请）；
2. 在智能测评的【效果测评 > 评测规则】已完成**发布**操作；
3. 在智能测评的【效果测评 > 效果评测任务】页面创建并成功运行至少一个测评任务。


## 操作指南
1. 登录 ADP，进入工作台；
2. 在左侧导航栏中，选择智能评测的【榜单】，进入榜单页面；
3. 系统将自动展示符合条件的榜单数据。如需将榜单数据用于离线分析，可点击页面顶部的【查看 PDF】按钮，下载完整的榜单报告。
