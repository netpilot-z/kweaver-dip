# 管理测试指标

## 概述
### 功能定义
测试指标用于量化评估被测对象在 Benchmark 场景下的性能、质量或效果表现，是对测试结果进行对比分析与客观判断的标准化度量。

## 操作步骤
1. 登录 ADP，在工作台左侧导航栏中选择智能评测模块下的【测试指标】选项。
    ○ **函数计算**：通过具体的量化指标（如关键词匹配率、ROUGE评分等）评估模型、自定义应用等性能。这些指标通常基于预定义的标准和算法，能提供客观的数值结果，但可能无法完全反映评估对象的实际应用效果。
    ○ **AI评价**：利用先进的语言模型模拟人类评估者的角色，对答案的质量进行自动评分，旨在提供快速、自动化的评估手段，同时保持一定的准确性和客观性。
2. 在【测试指标】页面的右上角，单击【上传】按钮。用户可根据页面提供的【示例模板】，上传所需的指标文件。
3. 指标文件上传完成后，可在当前页面直接查看以下内容：
    ○ 指标使用说明
    ○ 参数指标
    ○ 文件预览
