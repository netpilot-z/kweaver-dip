# 如何创建和使用 Decision Agent
模板功能允许用户将已创建的Decision Agent发布为可复用的模板，供其他用户快速创建具有相似功能的Agent。


### 创建模板
#### 操作步骤
1. **进入模板创建入口**
    - 登录ADP后，进入「开发」>「Decision Agent」页面
    - 找到您想要发布为模板的Decision Agent卡片，将鼠标悬停在卡片上
    - 点击卡片右上角的"···"操作菜单，选择【发布为模板】按钮  

2. **配置模板分类**
    - 点击【发布为模板】后，页面将弹出"发布模板"弹窗
    - 在弹窗中选择模板所属的分类  
    - 点击【确定】按钮，完成模板发布

3. **查看已发布模板**
    - 发布成功后，模板可在「广场」>「模板」菜单中查看  
    - 模板将按照所选分类显示在模板市场中，供所有用户使用


### 使用模板
#### 操作步骤
1. **进入模板库选择模板**
    - 登录ADP后，进入「广场」>「模板」页面，浏览或搜索可用的模板
    - 将鼠标悬停在目标模板卡片上，点击右上角"···"操作菜单，点击【使用模板创建】按钮  

2. **进入配置页面并自定义**
    - 点击【使用模板创建】后，系统将跳转至Decision Agent创建页面，并自动载入该模板的全部配置信息
    - 在该页面中，您可以查看模板预设的各项配置参数，所有配置项均支持自定义修改

3. **完成配置并发布**
    - 根据您的实际需求，修改或保留模板的配置项。编辑完成后，点击页面【发布】按钮


### 编辑模板
#### 操作步骤
1. **进入模板编辑入口**
    - 登录ADP后，进入「开发」>「我的模板」页面  
    - 找到您想要编辑的模板卡片，点击卡片进入编辑页面  

2. **修改模板配置**
    - 在模板编辑页面，您可以修改模板的所有配置项

3. **发布编辑后的模板**
    - 编辑完成后，有两种发布方式：
      - **方式一**：在编辑页面直接点击【发布】按钮，将模板重新发布到模板市场
      - **方式二**：点击【保存】按钮保存修改，然后在【我的模板】页面，将鼠标悬停在模板卡片上，点击"···"操作菜单中的【发布模板】按钮  


### 删除模板
#### 操作步骤
1. **进入模板删除入口**
    - 登录ADP后，进入「开发」>「我的模板」页面
    - 找到您想要删除的模板卡片，将鼠标悬停在卡片上

2. **执行删除操作**
    - 点击卡片右上角的"···"操作菜单，选择【删除】选项
    - 系统将提示确认删除，点击【确定】完成删除操作


### 常见问题
1. **Q1：哪些用户可以发布模板？**
    - A1：拥有 “模板发布权限” 的用户可以将自己创建的Decision Agent发布为模板。

2. **Q2：发布为模板后，原Decision Agent会受影响吗？**
    - A2：不会。发布为模板只是创建了一个模板副本，原Decision Agent仍然独立存在，可以正常使用和编辑。

3. **Q3：模板发布后，其他用户可以看到我的模板配置信息吗？**
    - A3：可以。模板发布至模板市场后，所有用户均可查看该模板的详细配置信息（包括角色指令、知识来源、技能设置等）。模板的核心价值在于其配置信息的复用性，用户可基于这些预设信息快速创建和自定义新的Agent。

4. **Q4：模板可以设置访问权限吗？**
    - A4：目前模板功能不支持设置访问权限，所有发布到模板市场的模板对所有用户可见。如需限制访问，请勿将Agent发布为模板。

5. **Q5：删除模板后，使用该模板创建的Decision Agent会受影响吗？**
    - A5：不会。删除模板不会影响已使用该模板创建的Decision Agent，这些Agent将独立存在，不受模板删除的影响。
