# Decision Agent 
Decision Agent 以 BKN 为基础，融合数据流处理能力，核心职能是完成智能体的创建、配置与全生命周期管理。通过整合多元知识来源、扩展技能模块、搭载智能模型及运用 Dolphin-Lang 编程语言，其可高效应对各类复杂决策任务，为业务场景提供智能化支撑。


## 核心组成结构
Decision Agent 的功能实现依赖四大核心模块，各模块分工明确、协同作用，共同构成其完整能力体系，具体详情如下表所示：

| 核心模块   | 定位                 | 具体内容                                                                 |
|------------|----------------------|--------------------------------------------------------------------------|
| 知识来源   | 智能体的 “血液”      | 提供智能体运行所需的基础信息，来源包括但不限于：<br>- 文档库（结构化 / 非结构化文档）<br>- BKN（领域内知识关联体系）<br>- 指标系统（业务核心数据指标）<br>- 知识条目（碎片化、精准化知识单元） |
| 技能       | 智能体的 “能力扩展器”| 突破基础功能边界，支撑复杂任务处理，包含：<br>- 技能 Agent（专项任务处理子智能体）<br>- MCP（模型上下文协议，保障模型交互一致性）<br>- 工具（外部功能调用组件，如数据查询工具）<br>- 算子（基础计算 / 处理单元，支撑数据运算） |
| 模型       | 智能体的 “大脑与思维中心” | 负责智能决策逻辑实现，核心特性：<br>- 多模型兼容：支持集成大模型（如通用对话模型）与小模型（如专项推理模型）<br>- 标准化接口：提供统一调用方式，降低模型接入与切换成本 |
| Dolphin-Lang | 智能体的 “神经系统”  | 专属流程配置语言，核心作用：<br>- 编排数据流（定义数据流转路径与处理顺序）<br>- 任务处理调度（分配任务至对应模块并监控执行）<br>- 决策逻辑落地（将业务规则转化为可执行流程） |


## 功能应用场景
Decision Agent 凭借其灵活的配置能力与强大的决策支撑能力，已广泛应用于多个领域，核心场景包括：

1. **决策支持系统**：为企业管理层提供数据驱动的决策建议，如市场趋势分析、风险评估等，降低决策主观性。
2. **业务流程自动化**：替代人工完成重复性、规则性决策环节，如订单审核、流程合规校验等，提升业务处理效率。
3. **智能客服**：通过整合业务知识与对话模型，实现用户问题的精准解答与需求预判，优化客户服务体验。
