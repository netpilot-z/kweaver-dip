# 创建 Decision Agent
本文档详细介绍了在 DIP 系统中创建、配置和发布 Decision Agent 的完整流程，包括前提条件、关键操作步骤、注意事项及常见问题解答，适用于开发平台用户参考。


## 前提条件
- 用户必须登录到系统并进入开发平台。
- 用户需要为 Decision Agent 配置必要的产品类型（例如：ChatBI、DIP、AnyShare 等）。
- 用户需要选择适当的数据源，如 BKN 或文档库，并进行配置。

## 创建 Decision Agent
### 操作步骤
1. 登录 ADP，点击「开发」>「Decision Agent」>【新建】按钮。  

### 新建 Decision Agent 配置页面说明
点击【新建】按钮后，系统将进入 Decision Agent 的创建与配置页面。该页面包含多个配置模块，需按以下步骤逐一设置

#### 基本信息
- **名称**：输入 Decision Agent 的名称，不超过 50 个字符。
- **简介**：简要描述此 Agent 的用途与功能，或由 AI 辅助生成，不超过 200 字。
- **头像**：可为 Agent 上传或选择标识头像，或由 AI 辅助生成。
- **所属产品**：选择此 Agent 关联的产品，如“DIP”。

#### 角色指令
支持两种配置模式：
- **自然语言模式**：按提示词格式撰写角色指令，或由 AI 辅助生成。
- **Dolphin（专家模式）**：可编写 Dolphin 代码，实现更复杂的任务与决策流程。

角色指令应包括：
- **角色定位**：明确 Agent 的身份与职责。
- **目标任务**：描述 Agent 需完成的主要任务。
- **执行约束**：设定权限、参数与行为边界。

#### 知识来源
配置 Agent 的知识库来源，支持：
- **BKN**：连接企业内部 BKN。
- **知识条目**：添加具体的知识条目，以丰富 Agent 的知识库。

#### 技能配置
从技能库中选择一个或多个技能模块，例如：
- 数据分析
- 合同匹配
- 检索增强
- 任务规划等

#### 决策模型
选择 Agent 执行任务时使用的 AI 模型，如：
- 大语言模型（通用对话与生成）
- 推理模型（结构化逻辑判断）

#### 高级配置（可选）
- **长期记忆**：配置是否保留对话历史。
- **默认开场**：设置启动时的问候语或初始提示，支持 AI 辅助生成。
- **相关问题**：配置 Agent 能够回答的相关问题列表。
- **任务规划**：设置任务分解与规划策略。
- **预设问题**：设置常见问题列表，方便用户快速选择，支持 AI 辅助生成。

#### 保存与发布
完成上述配置后，点击【保存】可存储为草稿；点击【发布】即可将 Agent 部署至平台，供用户调用。

##### 说明
1. 长期记忆功能：配置时可选择「启用长期记忆」，允许 Agent 记录并复用用户历史交互信息，提升决策精准度（如个性化推荐、上下文连贯响应）；
2. 配置校验：发布前建议逐项检查「输入配置、知识来源、技能模块」的正确性，避免因配置缺失或错误导致运行异常。


## 注意事项与边界说明
### 权限管理
- 仅拥有应用管理员权限或指定开发角色权限的用户，可执行「创建、编辑、发布、删除」Decision Agent 的操作；
- 普通用户仅具备「使用已发布 Agent」的权限，同时对自己创建的 Decision Agent 拥有编辑权限；
- 用户只能修改自己创建的未发布版本，已发布的 Agent 需通过版本管理进行更新。

### 发布限制
- 已发布的 Decision Agent 支持编辑修改，修改完成后可重新发布；
- 重新发布后，新版本将直接覆盖原有版本，线上环境将立即更新为最新版本；
- 所有历史版本均在 Decision Agent 的「配置信息」页面中完整记录，可供随时查阅；
- 如需回退至历史版本，可在「配置信息」中选择特定版本重新发布，该版本将作为最新版本上线。

### 数据隐私
- 启用「长期记忆模块」时，需遵守企业数据隐私政策及相关法规（如 GDPR、个人信息保护法等）；
- 定期审计长期记忆中的用户数据，避免敏感信息泄露或违规存储；
- 用户数据的存储和处理应符合企业安全规范，敏感数据需进行脱敏处理。

### 性能与稳定性
- 单个 Agent 同时处理任务数量存在上限，超出限制可能导致响应延迟；
- 复杂任务处理时长受模型性能、知识库大小等因素影响。


## 常见问题
Q1：**创建时无法选择合适的技能，怎么办？**
A1：可按以下步骤排查原因：
- 确认所需技能已在「技能模块库」中创建（未创建需先前往【技能管理】页面新建）。
- 检查技能是否已启用（未启用需在【技能管理】中开启「可用状态」）。
- 核实技能与当前 Decision Agent 的产品类型是否兼容（确认技能「适用产品范围」包含目标类型）。

Q2：**如何修改已发布的 Decision Agent？**
A2：已发布的 Agent 修改流程如下：
- 进入「Decision Agent」列表页，找到目标 Agent，点击 Agent 进入编辑。
- 完成配置后点击【发布】，系统自动更新为最新版本。
