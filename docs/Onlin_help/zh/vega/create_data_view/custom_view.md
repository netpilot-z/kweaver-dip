# 自定义视图

## 概述
自定义视图基于已创建的**原子视图**生成，支持对 1 个或多个原子视图的数据进行**过滤、去重**操作，生成满足特定业务需求的新视图。同时提供自定义视图的**导入、导出**及**分组管理**功能，帮助您高效组织和维护视图资源。


## 创建自定义视图
### 前提条件
在创建自定义视图前，请确保已完成以下操作：
1. 已在「 VEGA 虚拟化」>「 数据连接」中创建有效的数据连接。
2. 已在「 VEGA 虚拟化 」>「 数据视图 」>「 原子视图」中扫描数据连接，并成功同步至少 1 个原子视图。

### 具体步骤
#### 步骤 1：进入新建页面
1. 在「自定义视图」页面点击顶部【新建】按钮，弹出 “新建自定义视图” 窗口。
2. 填写基础信息：
   - **视图名称**：需唯一，建议包含业务场景（如 “2025Q3 活跃用户视图”）。
   - **所属分组**：从下拉列表选择已创建的分组（或选择 “未分组”，后续可调整）。
   - **描述**（可选）：填写视图用途（如 “用于分析 2025Q3 月活跃用户特征”）。
3. 点击【下一步】，进入可视化自定义视图的画布页面。

#### 步骤 2：添加原子视图
1. 在画布左侧工具栏点击【视图引用】图标，弹出「选择原子视图」窗口。
2. 在列表中勾选需要引用的原子视图（支持多选，如同时选择 “用户基础信息视图”和“用户行为日志视图”），点击【确定】。
3. 所选原子视图将以 “引用视图” 节点形式显示在画布中。

#### 步骤 3：配置数据过滤与去重
1. 点击画布中的【引用视图】节点，画布下方弹出「数据配置」面板。
2. 配置**数据过滤**（可选）：
   - 点击【添加过滤条件】，选择字段（如 “用户注册时间”）、运算符（如 “≥”）、值（如 “2025-07-01”），可添加多条件（支持 “且 / 或” 逻辑关联）。
   - 示例：筛选 “用户注册时间≥2025-07-01 且 用户等级 = VIP” 的数据。
3. 配置**数据去重**（可选）：
   - 勾选【启用去重】，选择去重字段（如 “用户 ID”），系统将按所选字段删除重复数据（保留第一条匹配数据）。
4. 点击【确定】，完成【引用视图】节点配置。

#### 步骤 4：连接节点并预览
1. 在画布中，点击【引用视图】节点的输出端口（右侧小圆圈），按住鼠标拖动至【输出视图】节点的输入端口（左侧小圆圈），释放鼠标完成连接（若添加多个【引用视图】节点，需依次连接至【输出视图】）。
2. 点击【输出视图】节点，画布下方弹出「字段预览」面板，查看最终视图的字段列表（包含所有【引用视图】节点的字段，自动去重重复字段名）。
3. 若需调整字段显示顺序，可在「字段预览」面板中拖动字段名调整位置。

#### 步骤 5：保存视图
1. 确认配置无误后，点击画布右上角【保存】按钮，系统成功提示 “保存成功”。
2. 自定义视图将显示在「自定义视图」列表中。

### 备选方案：导入自定义视图
若已通过其他环境导出自定义视图配置文件（JSON 格式），可直接导入：
1. 在「自定义视图」页面点击顶部【导入】按钮，弹出「导入视图」窗口。
2. 选择本地的 JSON 配置文件，点击【确定】。
3. 系统验证文件合法性（如字段映射、原子视图关联关系），验证通过后，视图将添加至指定分组。

### 查看视图详情
1. 在「自定义视图」列表中，点击目标视图名称，进入「视图详情」页面，可查看：
   - 基础信息：视图名称、分组、创建时间、更新时间。
   - 配置信息：引用的原子视图、过滤规则、去重字段。
   - 数据预览：前 100 条数据样例（支持按字段排序）。


## 常见问题
Q1：**创建自定义视图时，提示 “原子视图不存在” 怎么办？**
A1：请按以下步骤处理：
- 进入「原子视图」页面，确认目标原子视图是否存在。
- 若已删除，重新创建原子视图后再配置自定义视图。

Q2：**导出自定义视图后，导入到其他环境失败怎么办？**
A2：请按以下步骤排查：
- 检查目标环境的「原子视图」列表，确保已创建与源环境结构一致的原子视图。
- 若字段不一致，在源环境调整自定义视图的字段配置（如删除目标环境不存在的字段），重新导出后再导入。
