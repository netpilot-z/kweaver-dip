# 原子视图

## 概述
原子视图是通过扫描数据连接生成的虚拟视图，支持将不同类型的数据连接以**统一数据类型**呈现，有效降低数据使用门槛，提升数据可理解性，助力用户快速开展数据相关操作。

### 适用用户
- 数据管理员
- 数据分析师

### 重要限制
- 不支持通过扫描**内置索引库数据连接**新建原子视图，仅允许查看和预览此类数据连接生成的原子视图。
- 不支持扫描**Excel 类型数据连接**，若需基于 Excel 数据连接创建视图，需选中 Excel 数据连接后手动新建。


## 创建原子视图
### 前提条件
- 已完成数据连接配置（参考：如何创建数据连接？）。
- 已创建数据连接的扫描任务。

### 操作步骤
#### 配置并启动数据连接扫描
1. 登录系统后，在左侧导航栏中点击「数据连接」，进入「扫描管理」页面。
2. 在「扫描管理」页面，通过勾选数据连接列表的方式，选择需要扫描的数据连接范围（支持批量选择多个数据连接）。
3. 确认选择后，点击页面右下角的「开始扫描」按钮，启动数据连接扫描任务。

#### 查看扫描进度与结果
1. 扫描启动后，可在扫描任务详情查看当前扫描进度（包括已扫描数据连接数量、剩余数量）。
2. 扫描完成后，扫描出来的元数据信息会自动同步到原子视图。

#### 管理原子视图
1. 进入「数据视图」>「原子视图」，页面右侧将展示所有已扫描数据连接的原子视图及对应的扫描结果。
2. 针对单个原子视图，可执行以下操作：
    ○ **查看详情**：点击 “查看详情”，进入视图详情页，查看视图的完整配置与数据信息。
    ○ **编辑**：点击 “编辑”，修改视图的名称、描述等基础信息（数据连接关联信息不可修改）。
    ○ **数据预览**：点击 “数据预览”，查看视图对应的样本数据，验证数据准确性。
    ○ **删除**：点击 “删除”，在弹出的确认弹窗中点击 “确认”，删除该原子视图（删除后不可恢复，请谨慎操作）。
    ○ **查看详情**：点击 “查看详情”，进入视图详情页，查看视图的完整配置与数据信息。
    ○ **编辑**：点击 “编辑”，修改视图的名称、描述等基础信息（数据连接关联信息不可修改）。
    ○ **数据预览**：点击 “数据预览”，查看视图对应的样本数据，验证数据准确性。
    ○ **删除**：点击 “删除”，在弹出的确认弹窗中点击 “确认”，删除该原子视图（删除后不可恢复，请谨慎操作）。
