# 指标模型

## 概述
指标模型模块提供指标“研发 - 管理 - 使用”一体化能力，支持开发人员高效创建、管理指标，已创建的指标可作为智能体的知识来源。

支持三种指标类型：
- **原子指标**：基于业务事件的基础度量（如“支付金额”），不可拆分
- **衍生指标**：由原子指标 + 修饰词 + 时间周期构成（如“最近 1 天海外买家支付金额”）
- **复合指标**：由原子/衍生/其他复合指标叠加计算生成


## 创建指标模型
### 前提条件
在创建指标模型前，需完成以下操作：
1. 完成数据连接配置：进入【多模态数据湖 > VEGA 虚拟化 > 数据连接】，参考《如何创建数据连接》文档创建数据源
2. 完成数据视图配置：进入【多模态数据湖 > VEGA 虚拟化 > 数据视图】，参考《如何创建数据视图》文档创建数据视图


### 创建原子指标
原子指标基于事实表度量字段生成，**操作路径**：「VEGA 虚拟化」>「数据模型」>「指标模型」，点击【新建】，在下拉框中选择【原子指标】

#### 步骤 1：基本配置
| 配置项 | 说明 |
| --- | --- |
| 指标名称 | 输入具有业务含义的名称（如“订单支付金额”），必填 |
| ID | 可选，自定义唯一标识（若不填写，系统自动生成） |
| 分组 | 可选，选择指标所属业务分组（用于分类管理） |
| 标签 | 可选，添加关键词标签（如“支付业务”“核心指标”），便于检索 |
| 备注 | 可选，填写指标说明（如“统计所有支付渠道的订单金额总和”） |

填写完成后，点击【下一步】。

#### 步骤 2：模型配置
首先选择数据来源视图（需提前创建，参考《如何创建数据视图》），根据视图类型（VEGA 类/索引类）配置以下内容：

##### VEGA 类视图配置
| 配置项 | 操作说明 |
| --- | --- |
| 数据过滤 | 可选，开启后支持两种方式：<br>- SQL：在编辑器输入过滤 SQL 语句<br>- 字段限定：添加条件（如“缴费状态 = 已缴费”） |
| 度量计算 | 必选，选择度量方式：<br>- SQL 计算公式：输入聚合 SQL（如“SUM (pay_amount)”）<br>- 字段聚合：根据字段类型选择函数（参考下表） |
| 分析维度 | 可选，除默认日期时间维度外，添加/删除自定义维度标签 |
| 分组字段 | 可选，选择视图内用于分组的字段 |
| 结果排序 | 可选，设置排序优先级（基于分组字段） |
| 结果过滤 | 可选，添加结果集过滤条件（如“金额> 100”） |
| 日期时间标识 | 必选，选择视图中的日期/日期时间/时间戳类型字段（若无，需通过自定义视图转换类型） |
| 指标单位 | 必选，选择数值单位（如“元”“次”“个”） |

##### 索引类视图配置
| 配置项 | 操作说明 |
| --- | --- |
| 查询语言 | 必选，选择“PromQL”或“DSL” |
| 计算公式 | 必选，根据查询语言填写：<br>- PromQL：输入 PromQL 查询语句<br>- DSL：按以下规则配置（参考下方说明） |
| 度量字段 | 必选，当查询语言为 DSL 时：<br>- 若值聚合为 top_hits：填写_source 包含的数值字段<br>- 其他聚合：填写值聚合名称 |
| 指标单位 | 必选，选择数值单位（如“次”“GB”） |

**DSL 计算公式配置规则**：
- 必须包含 date_histogram（日期直方图）和 value_count（计数）聚合
- 支持嵌套聚合（层数≤7），不支持并行聚合
- 预输入 DSL 结构包含：过滤查询（size=0，不返回文档）、三层嵌套聚合（terms→date_histogram→value_count）
- 自定义 DSL 需遵循 OpenSearch 聚合语法，参考《OpenSearch Aggregations 文档》

#### 步骤 3：持久化配置（可选）
开启“持久化”开关，将指标数据存储到指定索引库，配置项：
- 索引库：选择目标索引库（需提前创建）
- 存储周期：设置数据保留时长（如“30 天”“永久”）
- 更新频率：设置数据同步频率（如“每小时”“每天”）

#### 步骤 4：完成创建
点击【确定】，返回指标模型列表。可点击目标指标操作栏的【数据预览】，通过筛选条件查看数据趋势。


### 创建衍生指标
衍生指标基于原子指标创建，**操作路径**：「VEGA 虚拟化」>「数据模型」>「指标模型」，点击【新建】，在下拉框中选择【衍生指标】

#### 步骤 1：基本配置（同原子指标）
#### 步骤 2：数据来源配置
- 选择基础原子指标：从已创建的原子指标列表中选择
- 数据过滤：同原子指标（支持 SQL/字段限定，可添加时间限定如“最近 7 天”）
- 结果排序/结果过滤：同原子指标
- 分析维度：继承原子指标维度，支持添加/删除自定义维度
- 指标单位：同原子指标

#### 步骤 3：完成创建（无持久化配置）
点击【确定】，支持通过【数据预览】查看结果。


### 创建复合指标
复合指标基于原子/衍生/其他复合指标创建，**操作路径**：「VEGA 虚拟化」>「数据模型」>「指标模型」，点击【新建】，在下拉框中选择【复合指标】

#### 步骤 1：基本配置（同原子指标）
#### 步骤 2：模型配置
- 引用指标：在左侧指标列表（原子/衍生/复合）中，点击目标指标添加到表达式编辑器
- 编辑计算表达式：使用运算符组合指标（如“指标 A - 指标 B”）
- 数据过滤/结果排序/结果过滤/分析维度：同原子指标
- 指标单位：根据计算逻辑选择（如“元”“百分比”）

#### 步骤 3：完成创建（无持久化配置）
点击【确定】，支持通过【数据预览】查看结果。


### 导入/导出指标模型
- **导入**：点击【导入】，选择 JSON 格式的指标配置文件（需符合系统模板格式）
- **导出**：在指标列表中，勾选目标指标，点击【导出】，生成 JSON 格式文件


## 注意事项
1. **指标依赖关系**：
   - 衍生指标必须基于原子指标创建，不可直接基于其他衍生指标
   - 复合指标不可脱离基础原子指标的维度范围，不可新增维度
2. **DSL 语法校验**：
   - 保存前系统自动校验 PromQL/DSL 语句合法性，不通过则需修改
   - 暂不校验字段权限，查询不存在的字段会返回空结果
3. **数据预览**：
   - 预览时需选择时间范围、维度筛选条件
   - 若数据量较大，预览可能存在延迟，建议缩小时间范围
