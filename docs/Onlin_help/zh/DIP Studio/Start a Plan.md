# 如何创建工作计划
工作计划是 **KWeaver DIP** 提供的一种主动式的智能项目协作能力，旨在帮助用户将长期目标拆解成可执行的任务，并通过数字员工自主决策、自动化推进任务执行、进度汇报和关键节点确认，实现工作目标的高效达成。通过本文，您可以了解如何轻松创建并管理工作计划。

## 创建工作计划的步骤
1. **进入数字员工会话页面**
在 **KWeaver DIP** 左侧导航栏，找到**"我的数字员工"**选项，按需选择数字员工，发起会话。

2. **输入目标**
在对话框中使用自然语言输入您的需求和目标，例如：
- "两周内学会数据分析"
- "半年内提升我的表达能力"
- "3 个月完成产品从 0 到 1 上线"

3. **确认计划**
数字员工将开始解析并拆解您的目标，如果您的目标不够明确，数字员工会通过多轮对话向您确认细节。确认无误后，数字员工将生成详细的执行路线图待您确认。确认后，点击**"生成计划"**，正式创建并执行工作计划。

## 工作计划的核心特性
- **智能目标拆解**：数字员工会根据您的需求，将长期目标拆解为可执行的阶段性任务，确保目标逻辑清晰、可操作。
- **自动化任务推进**：数字员工将自动推进任务的执行，无需用户手动干预，节省时间并提高执行效率。
- **节点化进度汇报**：在关键节点，数字员工会定期生成进度报告，并同步工作进展、潜在风险及后续计划。
- **交互式协作对齐**：数字员工会主动与您沟通确认，确保执行方向符合您的预期，及时调整任务规划。

## 工作计划管理
### 查看工作计划
您可以从以下两个入口找到工作计划：
- 入口一：在 **KWeaver DIP** 左侧导航栏，找到**"我的数字员工"**选项，点击某个数字员工后，可以查看该数字员工下所有的工作计划
- 入口二：在 **KWeaver DIP** 左侧导航栏，找到**"工作计划"**区域，点击【更多】按钮，进入**「工作计划」**列表，在该页面您可以查看不同数据员工创建的不同计划

找到工作计划后，您可以点击进入具体的工作计划详情页面，查看工作计划的执行记录和产出成果。

### 修改工作计划
进入工作计划的详情页面，在对话框中输入具体的修改指令，例如：“将第三阶段的目标调整为…”，等待数字员工确认后，即可完成工作计划的修改。

### 暂停或取消工作计划
1. **暂停计划**：在对话框中输入“暂停计划”或点击【…】按钮，在下拉选项中选择【暂停】按钮。
2. **取消计划**：输入“取消计划”，并选择**"仅暂停"**或**"确认终止"**。
