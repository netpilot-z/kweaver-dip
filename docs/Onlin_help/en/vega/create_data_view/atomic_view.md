# Atomic View

## Overview

An atomic view is a virtual view generated by scanning data connections. It can present different types of data connections in a **unified data type**, reducing the barrier to data usage, improving data understandability, and helping users start data-related work quickly.

### Target Users

- Data administrators
- Data analysts

### Important Limits

- You cannot create an atomic view by scanning a **built-in index library data connection**. Atomic views generated from such connections can only be viewed and previewed.
- **Excel-type data connections** cannot be scanned directly. If you need to create a view from an Excel data connection, select the Excel connection first and create the view manually.

## Create an Atomic View

### Prerequisites

- Data connections have been configured
- Scan tasks have already been created for the data connections

### Steps

#### Configure and Start Data Connection Scanning

1. After logging in, click **Data Connections** in the left navigation bar to open **Scan Management**.
2. On the **Scan Management** page, select the data connections to scan. Multiple connections can be selected in batch.
3. After confirming the selection, click **Start Scan** in the lower-right corner to start the scan task.

#### View Scan Progress and Results

1. After the scan starts, you can view the current progress in the scan task details, including how many data connections have already been scanned and how many remain.
2. After the scan is complete, the scanned metadata is automatically synchronized to atomic views.

#### Manage Atomic Views

1. Open **Data Views > Atomic Views**. The right side of the page shows all atomic views generated from scanned data connections together with their scan results.
2. For each atomic view, the following operations are supported:
   - **View Details**: open the detail page to inspect the full view configuration and data information
   - **Edit**: change basic information such as the view name and description. Data connection binding information cannot be modified
   - **Data Preview**: inspect sample data from the view to verify correctness
   - **Delete**: delete the atomic view after confirming in the dialog. This action cannot be undone
