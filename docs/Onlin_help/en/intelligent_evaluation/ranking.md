# Ranking

## Overview

The ranking page is a result view that sorts and displays evaluation outcomes based on unified test indicators and evaluation rules.

**Target users**: ADP administrators, data analysts, and evaluation task operators

## Usage Notes

- The ranking page only displays task results generated from **published** effectiveness evaluation configurations.
- Task results under unpublished configurations are not included in the ranking display.

## Prerequisites

Before using the **Ranking** feature, make sure the following preparations are complete:

1. You are logged in to ADP and have access to the **Intelligent Evaluation** module. If not, contact the platform administrator.
2. In **Intelligent Evaluation > Effect Evaluation > Evaluation Rules**, the target rule has already been **published**.
3. On the **Intelligent Evaluation > Effect Evaluation > Effect Evaluation Tasks** page, at least one evaluation task has been created and run successfully.

## Operation Guide

1. Log in to ADP and enter the workspace.
2. In the left navigation bar, choose **Ranking** under **Intelligent Evaluation** to enter the ranking page.
3. The system automatically displays eligible ranking data. If you need the data for offline analysis, click **View PDF** at the top of the page to download the complete ranking report.
