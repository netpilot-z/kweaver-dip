开发数字员工运营平台 DIP Studio 的后端服务

## 技术要求
- 使用 TypeScript 作为开发语言
- 使用 Express 作为 HTTP 服务框架
- 每个函数及参数都必须有文档注释
- 单元测试行覆盖率必须达到 90% 以上
